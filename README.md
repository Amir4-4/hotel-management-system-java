# Hotel Management System (Java)

##  Project Overview
This is a Hotel Management System developed using Java. The system is designed to manage hotel operations efficiently using Object-Oriented Programming (OOP) principles and Data Structures.

It provides a complete solution for handling guests, room reservations, billing, and service requests with a user-friendly graphical interface.

---

##  Features
- Guest management system
- Room booking and reservation handling
- Billing and payment processing
- Service request management
- Search functionality for guests and rooms
- Graphical User Interface (GUI)

---

##  Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Data Structures (Trees, Arrays, etc.)
- Searching Algorithms (Linear Search, Binary Search, A*)
- Java Swing / JavaFX (GUI)
- File Handling

---

##  Project Structure
- `Main.java` → Entry point of the application
- `HotelManagementSystem.java` → Core system logic
- `Guest.java` → Guest data model
- `Room.java` → Room management
- `Reservation.java` → Booking system
- `Billing.java` → Payment handling
- `HotelManagementSystemGUI.java` → User interface

---

## Algorithms Implemented
- Linear Search
- Binary Search
- A* Algorithm for optimization
